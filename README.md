A Chrome extension that shows today's Hijri date, prayer times, and upcoming sacred days. During Ramadan, it highlights suhoor and iftar.

## Features
- Hijri date + Gregorian date
- Prayer time table (Fajr, Dhuhr, Asr, Maghrib, Isha)
- Upcoming sacred days from the AlAdhan calendar
- Ramadan-focused UI
- Location via browser geolocation or city/country

## Run locally
1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this folder.
4. Open the extension and configure settings.

## Settings
- Location mode: browser geolocation or city/country
- Prayer time method
- Asr school
- Hijri date adjustment

## API
- Powered by AlAdhan: https://api.aladhan.com/
